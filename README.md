# Signolingo

## Sobre
Signolingo é um projeto desenvolvido para a disciplina de Libras no IFRN Natal Central. A intenção é promover o aprendizado do alfabeto e dos números na Língua Brasileira de Sinais.
Feito com Vue.Js
